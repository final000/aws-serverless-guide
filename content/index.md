# Serverless on AWS: Tool Installation Guide

Welcome to the setup guide for building serverless applications on AWS. Follow the sections in order to install and configure everything you need.
<!--th-->
# Serverless บน AWS: คู่มือติดตั้งเครื่องมือ

ยินดีต้อนรับสู่คู่มือการตั้งค่าสำหรับการสร้างแอปพลิเคชัน Serverless บน AWS ทำตามแต่ละหัวข้อตามลำดับเพื่อติดตั้งและตั้งค่าเครื่องมือทั้งหมดที่จำเป็น
<!--/-->

<!--en-->
## Prerequisites

- A computer running macOS, Linux, or Windows
- An AWS account ([create one here](https://aws.amazon.com/free/))
- Administrator access on your machine to install software
<!--th-->
## สิ่งที่ต้องมีก่อนเริ่มต้น

- คอมพิวเตอร์ที่ใช้ macOS, Linux หรือ Windows
- บัญชี AWS ([สร้างบัญชีที่นี่](https://aws.amazon.com/free/))
- สิทธิ์ Administrator บนเครื่องของคุณเพื่อติดตั้งซอฟต์แวร์
<!--/-->

<!--en-->
## What You'll Install

| Tool | Purpose |
|------|---------|
| AWS CLI | Interact with AWS services from your terminal |
| AWS SSO for CLI | Authenticate securely using your organization's identity provider |
| AWS SAM CLI | Build, test, and deploy serverless applications |
| Node.js | Runtime for Lambda functions and build tools |
| Python | Runtime for Lambda functions and SAM CLI dependency |
| Docker | Local testing and debugging with SAM CLI |
| Git | Version control for your projects |

## Serverless Blueprints

Once your tools are set up, check out the [Serverless Blueprints](blueprints.html) section for ready-to-use SAM template examples in Python, Node.js, and Java.

## เครื่องมือที่จะติดตั้ง

| เครื่องมือ | วัตถุประสงค์ |
|-----------|-------------|
| AWS CLI | ใช้งาน AWS services ผ่าน terminal |
| AWS SSO สำหรับ CLI | ยืนยันตัวตนอย่างปลอดภัยผ่าน identity provider ขององค์กร |
| AWS SAM CLI | สร้าง ทดสอบ และ deploy แอปพลิเคชัน serverless |
| Node.js | Runtime สำหรับ Lambda functions และ build tools |
| Python | Runtime สำหรับ Lambda functions และ dependency ของ SAM CLI |
| Docker | ทดสอบและ debug แบบ local ด้วย SAM CLI |
| Git | ระบบควบคุมเวอร์ชันสำหรับโปรเจกต์ |
